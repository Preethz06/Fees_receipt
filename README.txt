A interface to print the fees paid by students in a college. The login page contains admin_login and student_login which contains deleting,editing,viewing,adding student details and print student details respectively. There are two databases in MySQL created for information storing. One database contains admin username and password and the other database contains student details which has name,reg_no,dept,fees_paid,semester,dob and phone number. The login details for student login are his name and dob respectively.
